# 🩺 Endpoint Doctor

> Advanced Website & API Diagnostics Platform

Endpoint Doctor helps developers and businesses analyze websites, APIs, and backend endpoints to detect performance bottlenecks, rendering issues, network problems, and slow response times in real time.

---

## ✨ Features

### 🌐 Frontend Diagnostics
- Website speed analysis
- Render-blocking detection
- Image optimization checks
- Network waterfall analysis
- Caching inspection
- Core Web Vitals support

### ⚙️ Backend Diagnostics
- API endpoint testing
- Response time analysis
- Status code monitoring
- Payload size inspection
- SSL validation
- DNS lookup timing

### 🧠 Smart Insights
- Human-readable explanations
- Bottleneck detection
- Optimization recommendations
- Slow network simulation
- Real-world device testing

---

## 🚀 Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- Express.js
- Puppeteer
- Lighthouse
- Chrome DevTools Protocol

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/boi-network12/endpoint-doctor.git
```

---

# Frontend Setup

```bash
cd client
npm install
npm run dev
```

Frontend runs on:

```txt
http://localhost:3000
```

---

# Backend Setup

```bash
cd server
npm install
npm run dev
```

Backend runs on:

```txt
http://localhost:5000
```
